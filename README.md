* Performance profiling with < gprof >:
    + Compile the code(main.cpp) file witg the option: 
        -pg
    + Run the binary file as usual: 
        biofilm [OPTIONS ...]
    + Run gprof tool in order to analyze the performance:
        gprof ./bin/biofilm gmon.out | less
